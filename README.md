# bot-weather

Get the weather forecast

## Usage

```
Usage: weather [options] [location]

  Alias: wt

  Get the weather forecast

  Options:

    --help  output usage information
```

## Config (config.yaml)

```yaml
weather:
  token: your_api_token
  locations:
    - rennes
    - chateaubourg
```

## How do I get my api token?

Go to https://openweathermap.org/api
